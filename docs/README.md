# Documentacion TrikaWeb

Este directorio contiene la documentacion tecnica y operativa del proyecto.

## Indice

- `funcionalidades.md`: que hace el sistema, perfiles y modulos principales.
- `arquitectura.md`: stack, estructura del codigo, datos y seguridad.
- `flujos.md`: diagramas visuales de los flujos principales del sistema.
- `setup.md`: instalacion local, variables de entorno y base de datos.
- `api.md`: referencia de endpoints HTTP.
- `deploy.md`: despliegue en Vercel y checklist de release.

## Lectura recomendada

1. `setup.md`
2. `arquitectura.md`
3. `flujos.md`
4. `api.md`
5. `deploy.md`
